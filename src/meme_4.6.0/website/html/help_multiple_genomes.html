Multiple genomes

Some MEME Suite applications can utilize data from multiple genomes to increase the sensitivity of the analysis via comparative genomics. In this case, sequences in the primary genome, along with orthologous sequences from a set of comparative genomes, are used by the application in its final analysis.

When the multiple genomes option is available, a fixed set of comparative genomes is used by the MEME Suite application. You can view the identities of these genomes by clicking on "supported database" on the application's input web form. The types of sequences (e.g., length, location with respect to TSS etc.) are the same as for the primary genome.
